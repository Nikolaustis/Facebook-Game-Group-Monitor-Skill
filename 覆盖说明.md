# Facebook Group Monitor Skill V5.2.1 覆盖说明

## 本次修复

V5.2.1 将人工复核队列改为“先过数据门槛，再复核相关性”。弱相关候选只有同时满足以下条件才进入 `manual_review`：

- 成员数不少于 100；
- 今日帖子达到任务 `threshold`，或上周新增成员达到任务 `threshold`。

不达标的候选直接丢弃。人工复核表新增成员数、今日帖子和上周新增三列。

## 覆盖方法

1. 停止正在运行的第二轮任务。
2. 将压缩包解压到现有 Skill 根目录。
3. 选择替换同名文件。
4. 本补丁没有新增 npm 依赖，不需要重新执行 `npm install`。
5. 重新运行第二轮后生效；历史 Excel 不会自动重算。
