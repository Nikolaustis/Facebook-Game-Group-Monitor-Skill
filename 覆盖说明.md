# Facebook Group Monitor Skill V5.6.0 覆盖说明

## 覆盖方法

1. 停止正在运行的第一轮或第二轮任务。
2. 将本压缩包解压到现有 Skill 根目录，选择“替换同名文件”。
3. 不需要重新执行 `npm install`。
4. 删除旧 GeoNames 缓存：

```powershell
Get-ChildItem .\runs -Recurse -Filter "*geocode*cache*.json" | Remove-Item -Force
```

5. 重新运行第二轮。历史 XLSX 不会自动重算。

## 本版重点

- 多语种硬停用词与上下文受限地名。
- 游戏实体融合 token 整体清理。
- 多词地点不降级。
- 单 token GeoNames 结果行政层级/人口门槛。
- 高风险 ISO 国家码保护。
- `Hàn Quốc / LATHAM / GDL / SEQ Brisbane / Arab(s)` 等本地别名。

## GitHub 注意事项

不要上传：

```text
config/local/
runs/
node_modules/
*geocode*cache*.json
```

GeoNames 私有用户名仍应保存在 `config/local/geonames.local.json`。
