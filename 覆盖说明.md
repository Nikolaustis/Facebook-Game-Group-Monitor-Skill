# Facebook Group Monitor Skill V5.5.0 覆盖说明

## 覆盖方法

1. 停止正在运行的第一轮或第二轮任务。
2. 将压缩包解压到现有 Skill 根目录，选择替换同名文件。
3. 无新增 npm 依赖，不需要重新执行 `npm install`。
4. 建议清理旧 GeoNames 缓存后再运行第二轮：

```powershell
Get-ChildItem .\runs -Recurse -Filter "*geocode*cache*.json" | Remove-Item -Force
```

## 新地区顺序

```text
群名明确国家/地区/大区/本地城市
→ About Location 本地规则
→ About Location GeoNames
→ 高确定性语言映射
→ 群名 GeoNames 最后兜底
```

群名 GeoNames 只接受精确地名匹配，并过滤泰语、印尼语、马来语交易词、社群品牌词及所有已知游戏实体词。

## 针对样例的预期

- `LINE Idle Rangers ... Talk&Trade`：Thai → `TH`，不查询 `talk`。
- `... GREEN-TOWN`：Thai → `TH`，不查询 `green town`。
- `... @วิน`：Thai → `TH`，孤立泰语 token 不查询。
- `Pecinta Akun KUNING ... Jual/Beli`：Indonesian → `ID`，不查询 `jual`。
- `Cookierun Classic ซื้อขายรหัส`：若 About 为 Simla/Shimla，则 About GeoNames → `South Asia`；About 缺失时 Thai → `TH`。
- `Cookierun: Ovenbreak & Classic`：不把系列词作为 GeoNames query。
