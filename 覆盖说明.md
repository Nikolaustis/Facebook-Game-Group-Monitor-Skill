# Facebook Group Monitor Skill V5.2.0 覆盖说明

本包是基于 V5.1.0 的累计覆盖包，保留 GeoNames 自动启用、V5.0.1 endpoint 修复、About Location、蒙古语识别和强制关机功能，并新增地区识别精度修复。

## 本次解决的问题

- `™` 不再被识别为 `TM / Turkmenistan`。
- `grupo de pokemon go` 中的小写 `de` 不再被识别为德国；`Pokemon Go de Québec` 会优先识别 `Québec -> North America`。
- `台中`、`台南/臺南` 可直接识别为 `TW`。
- `Trójmiasto` 可识别为 `PL`，不会再被前缀 `Tr` 误判为 `TR`。
- `Danmark` 映射为 `EUR`。
- 韩语群名括号中的英文翻译及 `Come Together` 等泛词不会被送入 GeoNames。
- `Compra`、`Gift`、`trades`、`Bay`、`Only`、`Daily`、`Store`、`Level` 等非地点词会被安全过滤。
- 多词地名优先保留完整形式，例如 `San Diego`、`San Antonio`、`El Paso TX`、`Fort Worth`。

## 覆盖方法

1. 关闭正在运行的第一轮或第二轮任务。
2. 将本压缩包解压到现有 Skill 根目录，选择“替换同名文件”。
3. 不需要重新执行 `npm install`。
4. 建议删除旧地理缓存：

```powershell
Get-ChildItem .\runs -Recurse -Filter "*geocode*cache*.json" | Remove-Item -Force
```

5. 重新运行第二轮。历史 Excel 不会自动修正。

## 私有 GeoNames 配置

`config/local/geonames.local.json` 仍为本地私有配置，并由 `.gitignore` 忽略。上传 GitHub 时不要上传 `config/local/`。
