# Facebook Group Monitor Skill V5.3.0 覆盖说明

## 本次修改

1. GeoNames 查询前屏蔽当前游戏名称、别名、受控变体和高确定性 IP 根词，避免游戏词被当成城市或州。
2. 群名、About 用户文本和讨论区帖子进入语言识别前，屏蔽同一套游戏名称，避免英文标题干扰实际语言。
3. 暂存、正常最终、恢复最终三条 XLSX 链路的 K/L 列均固定为 `0.00%` 百分比格式。
4. GeoNames 缓存前缀升级为 `geonames-v5.3`。

## 覆盖方法

1. 关闭正在运行的第一轮或第二轮任务。
2. 将压缩包解压到现有 Skill 根目录，选择“替换同名文件”。
3. 不需要重新执行 `npm install`。
4. 重新运行第二轮；历史 XLSX 不会自动重算。

## 建议清理旧缓存

```powershell
Get-ChildItem .\runs -Recurse -Filter "*geocode*cache*.json" | Remove-Item -Force
```

V5.3.0 已使用新缓存命名空间，旧缓存不会被复用；清理仍有助于减小缓存文件并避免人工检查混淆。
