# V6.2.1 Validation

## Targeted regression cases

- `ONE PIECE BOUNTY RUSH ซื้อขาย แลกเปลี่ยน` => Thai => TH, even if sampled posts are English.
- `Seven Knights Re:Birth พูดคุย ซื้อขาย` => Thai => TH.
- `One Piece Bounty Rush Trade & Sell Việt Nam` => Vietnamese => VN, even if sampled posts are Arabic.
- `-ShutdownBefore 2026-07-17T09:00:00+08:00` implies shutdown-after-complete and is passed through manifest/runner/phase2 watcher.
- Shutdown watcher uses PowerShell and the full System32 shutdown path; Node no longer directly spawns shutdown.exe.
- Transient GeoNames failures retry; credential/account errors do not.
- Any fallback shutdown scheduled task is self-deleting.
